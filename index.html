códigos com git

  • git init - crie uma nova pasta, abra-a e execute o comando
  • git status - para ver o status do projeto
  • git add . - para adicionar tudo do index
  • git commit -m "comentários das alterações" - para confirmar a mudança
  • git push - para enviar alterações para seu repositório remoto
